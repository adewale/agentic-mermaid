Edit one node without regenerating the diagram

A render-only library makes an agent regenerate the whole diagram to move one node. Here, existing diagrams go parse → narrow → mutate → verify → serialize, and verify returns structured warnings in three tiers plus perceptual quality metrics.

Anything it cannot model round-trips losslessly through preserved source, so nothing is silently dropped.

Local by default

Run the library, the am CLI, or a local MCP server over stdio. Code Mode execute stays local, for trusted use only.

An optional hosted MCP exposes only render, verify, describe, and structured edits. Authless, rate-limited, no arbitrary code. Self-hosting is the recommended path.
