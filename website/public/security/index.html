Security and privacy

The editor stays browser-local; the hosted MCP is bounded and stateless.

Source stays in the browser for the editor. There is no REST render API. The hosted MCP at /mcp accepts JSON-RPC only, caps request bodies at 128KB and tool inputs at 64KB, and runs Code Mode execute in an isolated on-demand Worker with network access disabled, an empty environment, and a CPU budget.

Calling the hosted MCP means your diagram source or code travels to this site’s server, and successful responses are edge-cached for up to a day. If a diagram must stay local, use the library, the CLI, or a self-hosted MCP server — rendering and verification never require the network.
