# Product

## Product Purpose
Agentic Mermaid is an open-source Mermaid rendering and editing toolkit, forked from `lukilabs/beautiful-mermaid`, for turning diagram source into polished ASCII, PNG, and SVG outputs, Unicode terminal diagrams, and agent-verifiable structured edits. It is published as `agentic-mermaid`, maintained in `adewale/beautiful-mermaid`, and focused on expanded diagram coverage, semantic role-based styling, export tools, an agent-native safe-edit workflow, and a GitHub Pages-hosted showcase/editor.

## Register
product

## Users
- Developers evaluating whether the renderer supports their Mermaid syntax and export needs.
- Technical writers and maintainers who need clean ASCII, PNG, and SVG diagrams for docs, READMEs, PDFs, and terminals.
- Designers and product engineers exploring semantic role styling across diagram families.

## Primary Jobs
- Paste or write Mermaid syntax and see a reliable rendered preview.
- Discover supported diagram families and fork-specific capabilities quickly.
- Export or share rendered diagrams without hidden affiliation or deployment assumptions.
- Compare many rendering samples without overwhelming the page.

## Tone
Precise, calm, technical, and transparent. The interface should feel like a dependable developer tool, not a marketing funnel.

## Strategic Principles
- Blank editor by default, with obvious example discovery.
- Familiar product UI over decorative novelty.
- Dense where useful, readable where explanatory.
- Semantic styling and fork differences should be discoverable without implying upstream or Craft affiliation.
- Accessibility, mobile behavior, and export/error states are part of the product, not polish afterthoughts.

## Anti-References
- Generic AI/SaaS gradient hero pages.
- Hidden or blocked onboarding.
- Desktop-only editor layouts.
- Export controls that appear successful when no diagram exists.
- Branding or deployment copy that implies Craft ownership of this fork.
