```mermaid
flowchart TD
  User((User)) --> Login[Login Page]

  MFA -- Yes --> Code[Enter MFA Code]
  Code --> CodeCheck{Code valid?}
  CodeCheck -- Invalid --> Code
  CodeCheck -- Valid --> Session[Create Session]
```
